# Meridian — Private Banking UI

A premium private banking web application UI with a dark gold theme, built entirely with HTML, CSS, and JavaScript — no frameworks, no dependencies.

## Features

- **Overview Dashboard** — balance display, quick stats, recent transactions
- **Cards Management** — freeze/unfreeze card, set monthly & daily limits
- **Transfers** — send money with recipient management 
- **Analytics** — spending charts and breakdowns
- **Activity Log** — full transaction history with filters
- **Goals** — savings goal tracking with progress bars
- **Notifications** — real-time notification center
- **Concierge** — private banking concierge drawer
- **Dark/Gold Theme** — luxury aesthetic with CSS custom properties
- **Local Storage DB** — client-side data persistence via `MeridianDB`
- **Responsive** — works on desktop and mobile

## Tech Stack

| Layer      | Tech                        |
|------------|-----------------------------|
| Markup     | HTML5                       |
| Styling    | CSS3 (custom properties, grid, flexbox) |
| Logic      | Vanilla JavaScript (ES6+)   |
| Storage    | localStorage (`MeridianDB`) |
